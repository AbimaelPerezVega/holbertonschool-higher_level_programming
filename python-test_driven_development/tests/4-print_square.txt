==============================
How to Use 4-print_square.py
==============================

This module defines a function ``print_square(size)`` to print a square with the character #.

Usage
=====

``print_square(size)`` prints a square with the specified size using the character #.

::

    >>> print_square = __import__('4-print_square').print_square
    >>> print_square(4)
    ####
    ####
    ####
    ####

::

    >>> print_square(10)
    ##########
    ##########
    ##########
    ##########
    ##########
    ##########
    ##########
    ##########
    ##########
    ##########

::

    >>> print_square(0)
    (no output)

::

    >>> print_square(1)
    #

If the size is less than 0, a ValueError exception is raised:

::

    >>> print_square(-1)
    Traceback (most recent call last):
    ValueError: size must be >= 0

If the size is not an integer, a TypeError exception is raised:

::

    >>> print_square(1.5)
    Traceback (most recent call last):
    TypeError: size must be an integer
